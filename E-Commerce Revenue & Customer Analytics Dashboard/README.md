# E-Commerce Sales & Customer Analytics Dashboard

## Application Preview

![E-Commerce Revenue & Customer Analytics Dashboard](Dashboard_Screenshot.png)
## Project Overview

This project analyzes e-commerce sales data to generate business insights using an interactive dashboard. Data cleaning and preprocessing were performed using Python, followed by data analysis in SQL and visualization in Power BI.

The dashboard enables stakeholders to monitor sales performance, customer behavior, and geographic revenue distribution to support data-driven decision-making.

* SQL

* Data Cleaning

---

## Project Workflow

1. Cleaned and transformed raw sales data using Python
2. Loaded the cleaned dataset into a SQL database
3. Performed data analysis using SQL queries
4. Built an interactive dashboard in Power BI
5. Generated business insights and KPI reports

---

## Key Features of the Dashboard

* Total Revenue KPI
* Total Orders KPI
* Total Customers KPI
* Top 10 Products by Revenue
* Monthly Revenue Trends
* Revenue by Country
* Repeat vs One-time Customer Distribution
* Interactive Filters (Country and Date)

---

## Key Insights

* Processed and analyzed **18K+ transactions**
* Identified **November 2011** as the highest revenue month
* Repeat customers contributed the majority of purchases
* United Kingdom generated the highest revenue
